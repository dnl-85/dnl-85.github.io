le zen du C :

les programmes ici présents peuvent être compilés tel quel, et les commentaires inclus dans chacun d'eux permettent d'en expliquer le fonctionnement.

étant un utilisateur Linux Mint (noyau Debian), j'utilise gcc afin de compiler mes programmes.


		  	pour installer gcc depuis $bash :
			sudo apt-get install gcc
			
			la syntaxe pour compiler avec gcc depuis $bash :
			gcc programme_source.c -o programme_executable -lm
			
			note: l'option -lm permet d'intégrer la lib' math.h au programme, si vous vous en servez.
